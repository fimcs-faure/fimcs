EN COURS

Design

La gestion de toute l'apparence du site (menus, données etc). Cela inclut aussi les catégories remplies de manière adaptée.

à faire

Créer les pdf des cours

Dans un premier temps, tous les cours ayant eu lieu depuis le début de l'année seront disponibles au format pdf. Ils seront ensuite mis de manière plus esthétique dans le site.

à faire

Architecture

Toute l'architecture du site doit être pensée pour permettre une meilleure navigation. C'est un travail invisible mais nécessaire.

à faire

Base de données

Une base de données doit être conçue pour permettre une meilleure optimisation du site. Cela implique par exemple de pouvoir s'inscrire à un cours bonus sans passer par Google Forms.

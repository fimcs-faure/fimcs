CHAPITRE 1 - LE LANGAGE PYTHON

I)   Algorithme, machine :

Un algorithme est une suite d’instructions ayant pour but de produire un résultat (calcul, image…).

On s’intéresse en SNT aux algorithmes programmés sur des machines.

Une machine en SNT comporte :

  • Un processeur qui fait les calculs
  • Une mémoire (disque dur, RAM) qui conserve les données
  • Une interface qui permet à l’utilisateur d’entrer des données (clavier, souris, écran tactile…) et à la machine de communiquer des données (écran, écouteurs)

Algorithme Langage Python

Tracer un polygone régulier convexe à n côtés :

def polygone(l,n) :
      turn=360/n
      for i in range(n):
            turtle.forward(l)
            turtle.left(turn)
polygone(100,5)
